### Setup

> buildscript {  
>     // ...  
>     ext.kotlin_version = '<version to use>'  
>  
>     dependencies {  
>         classpath "org.jetbrains.kotlin" +  
>             "kotlin-gradle-plugin:$kotlin_version"  
>     }  
> }  
>    
> apply plugin: 'kotlin'  
>    
> dependencies {  
>     compile "org.jetbrains.kotlin:kotlin-stdlib-jre8"  
> }
