Mobile Tester Agent

AI-Powered Test Automation

Featured Project

An innovative AI agent for testing Android, iOS, and Web apps built with Koog. This project leverages LLM-based agents (Gemini, GPT, LLaMA) to automate testing through natural language scenarios, making it accessible for developers and testers of all levels. It executes tests on Android Emulator or physical devices via ADB, generates comprehensive reports, and provides a modern web dashboard for test configuration and result visualization.

Tech Stack: Kotlin, Ktor, Koog AI, ADB, Android SDK, React (Frontend)

Nomad Ecosystem

Decentralized Application Suite

Blockchain Project

A comprehensive suite of blockchain applications demonstrating full-stack decentralized development. The ecosystem includes nomad-app (Android application for interacting with smart contracts), nomad-core (Node.js backend for blockchain interactions), and nomad-token (Solidity smart contracts for NomadBadge and NomadRewardToken). This project showcases the complete journey from smart contract development on Ethereum to mobile user interfaces, featuring badge systems, token economics, and seamless web3 integration.

Tech Stack: Android, Kotlin, Node.js, Solidity, Ethereum, Web3.js

InsureBlox

Blockchain-Based Insurance

Chainlink Hackathon

A groundbreaking parametric insurance solution for vessels navigating rough seas, developed for the Chainlink hackathon. This innovative DeFi project utilizes blockchain technology and Chainlink oracles to create transparent, automated insurance processes based on real-world weather and sea condition data. The solution eliminates traditional claim disputes through smart contract automation and provides instant payouts when predetermined conditions are met.

Tech Stack: Solidity, Chainlink Oracles, DeFi Protocols, Smart Contracts
